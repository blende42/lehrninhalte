# Übungen – String-Klasse (Vertiefung)

## 1. Methoden kombinieren

```java
String input = "  Hallo Welt  ";
```

**Auftrag:**
- Leerzeichen entfernen
- alles klein schreiben
- Ergebnis ausgeben

---

## 2. Teilstring

```java
String text = "Java Programmierung";
```

**Auftrag:**
- "Java" extrahieren
- "Programmierung" extrahieren

---

## 3. Text prüfen

```java
String text = "Ich lerne Java";
```

**Auftrag:**
- prüfen, ob "Java" enthalten ist
- prüfen, ob "Python" enthalten ist

---

## 4. Ersetzen

```java
String text = "Banane";
```

**Auftrag:**
- alle "a" durch "o" ersetzen

---

## 5. Eingabe validieren

```java
Scanner scanner = new Scanner(System.in);
String input = scanner.nextLine();
```

**Auftrag:**
- Eingabe trimmen
- prüfen ob leer
- falls leer → Fehlermeldung

---

Input:
hallo welt
