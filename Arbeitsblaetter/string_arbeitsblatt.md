---

## Theorie

### Was ist ein String?

```java
String text = "Hallo";
```

👉 Strings sind Objekte

---

### Vergleich

```java
String a = "Hallo";
String b = "Hallo";
```

- `==` → Referenz
- `.equals()` → Inhalt

---

### Strings verändern

```java
String text = "Hallo";
text.toUpperCase();
```

👉 erzeugt neuen String

---

### Wichtige Methoden

```java
text.toUpperCase()
text.toLowerCase()
text.trim()
text.contains("Java")
text.length()
```
