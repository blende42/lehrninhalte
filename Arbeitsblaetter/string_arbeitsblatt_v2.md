# Tag 3 – String-Klasse (Vertiefung)

## Lernziele

- Ich kann Strings gezielt bearbeiten und kombinieren
- Ich verstehe, wie mehrere String-Operationen zusammenwirken
- Ich kann Eingaben validieren und weiterverarbeiten
- Ich erkenne häufige Fehlerquellen und kann sie vermeiden

---

## Theorie

### Kombination von Methoden

```java
String input = "  Hallo Welt  ";
String result = input.trim().toLowerCase();
```

👉 Methoden können kombiniert werden

---

### Wichtige neue Methoden

```java
text.substring(0, 5)
text.contains("Java")
text.replace("a", "o")
```

---

### Vergleich (Reminder)

```java
text.equals("Hallo")
```

👉 immer equals verwenden
