# SVG Workflow Sheet (EFZ Applikationsentwicklung)

## 🎯 1. Entscheidung

Was willst du darstellen?

- [ ] Vergleich → Vergleichs-Template  
- [ ] Ablauf → Prozess-Template  
- [ ] System → Architektur-Template  
- [ ] Konzept → Konzept-Template  

---

## ⚙️ 2. Prompts

### 🔀 Vergleich

```
Hier ist eine SVG-Vorlage.

Aufgabe:
Fülle diese Vorlage inhaltlich aus.

Thema:
[THEMA]

Linke Seite:
[INHALT]

Rechte Seite:
[INHALT]

Zusammenfassung:
[4 Stichpunkte]

WICHTIG:
- Struktur beibehalten
- nur Inhalte ersetzen
- Layout nicht verändern
```

---

### 🔄 Prozess

```
Hier ist eine SVG-Vorlage.

Thema:
[PROZESS]

Schritte:
1. [...]
2. [...]
3. [...]

Zusatz:
[Fehler / Hinweise]

WICHTIG:
- linearer Ablauf
- keine Layoutänderung
```

---

### 🏗️ Architektur

```
Hier ist eine SVG-Vorlage.

Thema:
[ARCHITEKTUR]

Ebenen:
- Client
- Backend
- Daten

Zusatz:
[CI/CD, Monitoring]

WICHTIG:
- Struktur beibehalten
- klare Schichten
```

---

### 🧠 Konzept

```
Hier ist eine SVG-Vorlage.

Thema:
[KONZEPT]

Kern:
[Definition]

Aspekte:
- [...]
- [...]

Merksatz:
[...]

WICHTIG:
- zentrales Element
- klare Struktur
```

---

## 🔍 3. Qualitäts-Check

- [ ] Kein Text abgeschnitten  
- [ ] Keine Überlappungen  
- [ ] Genügend Abstand  
- [ ] Lesbar auf A4  
- [ ] Inhalt korrekt  

---

## 🚀 Anwendung

1. Template wählen  
2. Prompt ausfüllen  
3. SVG generieren  
4. Check durchführen
